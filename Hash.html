INSERT

HMSET

同时将多个 field-value (域-值)对设置到哈希表 key 中

redis> HMSET website google www.google.com yahoo www.yahoo.com
OK

redis> HGET website google
"www.google.com"

redis> HGET website yahoo
"www.yahoo.com"

HSET

将哈希表 key 中的域 field 的值设为 value

redis> HSET website google "www.g.cn" # 设置一个新域
(integer) 1

redis> HSET website google "www.google.com" # 覆盖一个旧域
(integer) 0

HSETNX

将哈希表 key 中的域 field 的值设置为 value ，当且仅当域 field 不存在

redis> HSETNX nosql key-value-store redis
(integer) 1

redis> HSETNX nosql key-value-store redis # 操作无效，域 key-value-store 已存在
(integer) 0

DELETE

HDEL

删除哈希表 key 中的一个或多个指定域，不存在的域将被忽略

redis> HGETALL abbr # 测试数据
1) "a"
2) "apple"
3) "b"
4) "banana"
5) "c"
6) "cat"
7) "d"
8) "dog"

redis> HDEL abbr a # 删除单个域
(integer) 1

redis> HDEL abbr b c # 删除多个域
(integer) 2

redis> HGETALL abbr
1) "d"
2) "dog"

UPDATE

HINCRBY

为哈希表 key 中的域 field 的值加上增量 increment

redis> HEXISTS counter page_view
(integer) 0

redis> HINCRBY counter page_view 200 # increment 为正数
(integer) 200

redis> HGET counter page_view
"200"

redis> HINCRBY counter page_view -50 # increment 为负数
(integer) 150

redis> HGET counter page_view
"150"

HINCRBYFLOAT

为哈希表 key 中的域 field 加上浮点数增量 increment

redis> HSET mykey field 10.50 # 值和增量都是普通小数
(integer) 1

redis> HINCRBYFLOAT mykey field 0.1
"10.6"

SELECT

HEXISTS

查看哈希表 key 中，给定域 field 是否存在

redis> HEXISTS phone myphone
(integer) 0

redis> HSET phone myphone nokia-1110
(integer) 1

redis> HEXISTS phone myphone
(integer) 1

HGET

返回哈希表 key 中给定域 field 的值

redis> HSET site redis redis.com # 域存在
(integer) 1

redis> HGET site redis
"redis.com"

redis> HGET site mysql # 域不存在
(nil)

HGETALL

返回哈希表 key 中，所有的域和值

redis> HSET people jack "Jack Sparrow"
(integer) 1

redis> HSET people gump "Forrest Gump"
(integer) 1

redis> HGETALL people
1) "jack"
2) "Jack Sparrow"
3) "gump"
4) "Forrest Gump"

HKEYS

返回哈希表 key 中的所有域

redis> HMSET website google www.google.com yahoo www.yahoo.com # 哈希表非空
OK

redis> HKEYS website
1) "google"
2) "yahoo"

HLEN

返回哈希表 key 中域的数量

redis> HSET db redis redis.com
(integer) 1

redis> HSET db mysql mysql.com
(integer) 1

redis> HLEN db
(integer) 2

redis> HSET db mongodb mongodb.org
(integer) 1

redis> HLEN db
(integer) 3

HMGET

返回哈希表 key 中，一个或多个给定域的值

redis> HMSET pet dog "doudou" cat "nounou" # 一次设置多个域
OK

redis> HMGET pet dog cat fake_pet # 返回值的顺序和传入参数的顺序一样
1) "doudou"
2) "nounou"
3) (nil)

HVALS

返回哈希表 key 中所有域的值

redis> HMSET website google www.google.com yahoo www.yahoo.com
OK

redis> HVALS website
1) "www.google.com"
2) "www.yahoo.com"
